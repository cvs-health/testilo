Introduction

This is a digest of results from BATCH (Battery of Accessibility Tests for CVS Health).

BATCH (a product of the CVS Health Accessibility Solutions Team) runs 1230 automated accessibility tests drawn from nine different packages: Alfa, Axe, Continuum, Equal Access, HTML CodeSniffer, Nu Html Checker, Tenon, Testaro, and WAVE.

BATCH ran these tests on the web page named above and gave the page a score of __totalScore__, where 0 would be perfect. For perspective, web pages belonging to CVS Health have received scores in the range of 100 to 18,000.
