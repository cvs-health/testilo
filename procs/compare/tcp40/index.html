Introduction

The table below compares __pageCount__ web pages on accessibility. The page names are links to the pages on the web. The scores are links to digests that explain in detail how the scores were computed.

The pages were tested by Testaro. Testaro used ten tools (Alfa, ASLint, Axe, Editoria11y, Equal Access, HTML CodeSniffer, Nu Html Checker, QualWeb, Testaro, and WAVE) to perform about 900 automated accessibility tests.

Testilo classified the problems found by these tests into issues and assigned a score to each page. A perfect score would be 0. Higher scores indicate more issues, more instances of them, more serious issues, and more of the tools reporting instances of issues.
