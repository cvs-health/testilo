Accessibility score comparison

Introduction

The tsp10 accessibility testing procedure was executed by Testaro on 1 web pages. The procedure performed 808 tests on each page. Of these, 16 tests are custom tests defined by Testaro, and the others belong to packages of tests created by others. Testaro produced reports enumerating the test results.

These tests—like all tests—are fallible. The failures described in the reports merit investigation as potential opportunities for improved accessibility. But some reported faults may not actually harm accessibility, and some accessibility faults can escape detection by any of these tests.

Given the reports produced by Testaro, another procedure, tsp10a, assigned weights to the tests and computed a total score for each page (where 0 is the best possible score), adding a section on scoring to each report. Just as tests are fallible, any scoring procedure is subjective, and different reasonable procedures could assign different scores on the basis of the same report.

A third procedure, tdp10a, created a digest from each report, summarizing the tests and how the score was computed. The report for each page is appended to the page’s digest.

This page, produced by the ttp0 procedure, lists the tested web pages, in order of their scores (best to worst). The comparative table includes links to the pages and to the digests.
