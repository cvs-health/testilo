Introduction

The table below compares __pageCount__ web pages on accessibility. The page names are links to the pages on the web. The scores are links to digests that explain in detail how the scores were computed.

The pages were:

  1. Tested by Testaro with procedure __scriptID__
  2. Scored by Testilo with procedure __scorer__
  3. Digested by Testilo with procedure __digester__
  4. Compared by Testilo with procedure __comparer__

Testaro used nine tools (Alfa, Axe, Continuum, Equal Access, HTML CodeSniffer, Nu Html Checker, Testaro, and WAVE) to perform about 920 automated accessibility tests. Testilo used its scoring procedure to assign a score to each page, with 0 being perfect.
