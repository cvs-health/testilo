  1. Tested by Testaro with procedure __scriptID__

Testaro used nine tools (Alfa, ASLint, Axe, Equal Access, HTML CodeSniffer, Nu Html Checker, Testaro, and WAVE) to perform about 960 automated accessibility tests. Testilo assigned a score to each page, with 0 being perfect.
